what is the difference between inline and block elements in HTML?

Block elements always start on a new line and take up the full width available. For example, div, h1, p, form, header, footer, section, article, aside, nav, ul, ol, li, table, tr, td, th, tbody, thead, tfoot, blockquote, pre, hr, and more.

Inline elements do not start on a new line and only take up as much width as necessary. For example, span, a, strong, em, img, br, input, button, label, select, textarea, and more.
